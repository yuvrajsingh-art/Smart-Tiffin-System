# 🍱 Mess / Tiffin Management System

## 📘 **Admin Full Documentation (Design/UI Excluded)**

> **Project Type:** Real-life Production Project
> **Admin Model:** Single Admin (Owner / Operator)
> **Scope:** Customer + Provider + Operations + Finance + System

---

## 👑 1. Admin Overview

**Admin** system ka **single decision-maker** hota hai. Admin daily operations, customers, providers (kitchen/mess), payments aur reports ko manage karta hai.

**Admin Responsibilities:**

* Business operations chalana
* Customers handle karna
* Providers (kitchen/mess) coordinate karna
* Paisa & billing manage karna
* Data accuracy & security ensure karna

---

## 🔐 2. Authentication & Security

### Admin Actions:

* Login / Logout
* Password change
* Forgot & reset password
* Secure session/JWT handling
* Login activity tracking

**Purpose:**
Unauthorized access rokna aur data secure rakhna.

---

## 👥 3. Customer Management

### Admin Can:

* New customer add karna
* Customer details update (name, phone, address, area)
* Customer activate / deactivate
* Subscription pause / resume
* Date-wise absence mark
* Customer profile view
* Customer order history dekhna
* Customer payment history track karna

**Real-life Example:**
Customer bole: “Kal tiffin nahi chahiye” → Admin absence mark karta hai.

---

## 🏭 4. Provider Management (Mess / Kitchen)

> **Provider = Food preparation unit (kitchen/mess)**

### Admin Can:

* Provider details manage
* Provider active / inactive karna
* Daily capacity set (kitne tiffin ban sakte hain)
* Menu assign karna
* Preparation status track karna

**Purpose:**
Food supply aur demand ka balance maintain karna.

---

## 📦 5. Tiffin Plans & Subscriptions

### Admin Controls:

* Weekly / Monthly plans banana
* Veg / Non-Veg options
* Lunch / Dinner selection
* Plan price set karna
* Total days define karna
* Custom plan assign karna
* Subscription pause / resume
* Auto expiry & renewal manage karna

**Example:**
Monthly Veg Lunch – ₹2600 (26 Days)

---

## 🍛 6. Daily Menu Management

### Admin Tasks:

* Daily menu create/update
* Weekly menu plan
* Special dish mark
* Holiday ke din menu disable
* Menu history maintain

**Purpose:**
Daily food planning aur transparency.

## 📋 7. Orders / Daily Tiffin Generation

* Active subscriptions se daily tiffin auto-generate

### Admin Can:

* Today’s orders view
* Order status update (Pending → Prepared → Delivered)
* Order list re-generate
* Delivery list print/export

Service quality aur accountability.
